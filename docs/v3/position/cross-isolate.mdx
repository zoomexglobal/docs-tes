Select cross margin mode or isolated margin mode per symbol level
> **Normal account covers: USDT perpetual / Inverse contract**

### HTTP Request
POST `/cloud/trade/v3/position/switch-isolated`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Product type <ul><li>Normal account: `linear`, `inverse`. _Please note that `category` is **not** involved with business logic_</li></ul> |
|symbol |**true** |string |Symbol name |
|tradeMode |**true** |integer |`0`: cross margin. `1`: isolated margin |
|buyLeverage |**true** |string |The value must be equal to `sellLeverage` value|
|sellLeverage |**true** |string |The value must be equal to `buyLeverage` value|


---

### Response Parameters
None

X-BAPI-API-KEY: XXXXX

    "category": "linear",
