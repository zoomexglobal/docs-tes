### 2025-12-24

1. After upgrading UTA, users need to apply for a new API key to call the interface
2. After upgrading UTA, use this interface to query the pre-upgrade history: [Get Pre-upgrade Order History](pre-upgrade/pre-upgrade-order-list#)
3. After upgrading UTA, check the transaction record before the upgrade using this interface: [Get Pre-upgrade Trade History](pre-upgrade/pre-upgrade-execution#)
4. After upgrading UTA, use this interface to check the profit and loss of position squaring before upgrading: [Get Pre-upgrade Closed PnL](pre-upgrade/pre-upgrade-close-pnl#)
5. **Trade - Place Order:** `qty`: Spot, You can represent the unit of the market order qty by setting `marketUnit`. The market order is `quoteCoin` by default, and the market order is `baseCoin` by default
6. **Trade-Get Order History:** The queryable status and range have been adjusted, refer to: [Query historical orders](order/order-list#)
7. **Position - Set leverage:** UTA double position mode, `buyLeverage` must be equal to `sellLeverage`
8. **Assets - Get Transferable Amount:** A new interface has been added to query the amount that can be transferred from a trading account
9. **Assets - Get Wallet Balance:** Enter `UNIFIED` to check the balance of the trading account
