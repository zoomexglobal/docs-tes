Query recent public trading data in Bybit.
> **Covers: USDT perpetual/ Inverse contract**

You can download archived historical trades here:
* <a href="https://www.zoomex.com/trade/usdt/">USDT Perpetual</a>
* <a href="https://www.zoomex.com/trade/inverse/">Inverse Perpetual</a>

### HTTP Request
GET `/cloud/trade/v3/market/recent-trade`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `linear`,`inverse` |
|[symbol](../enum#symbol) |false |string |Symbol name|
|limit |false |integer |Limit for data size per page.<ul><li>[1,1000], default: `500`</li></ul>|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Products category |
|list |array |Object |
|> execId |string |Execution ID |
|> symbol |string |Symbol name |
|> price |string |Trade price |
|> size |string |Trade size |
|> side |string |Side of taker `Buy`, `Sell`|
|> time |string |Trade time (ms) |


---

### Request Example

```shell
GET /cloud/trade/v3/market/recent-trade?category=spot&symbol=BTCUSDT&limit=1 HTTP/1.1
Host: openapi-testnet.zoomex.com
```
