Query asset information

:::info
For now, it can query `SPOT` **only**.
:::

### HTTP Request
GET `/cloud/trade/v3/asset/transfer/query-asset-info`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](../enum#accounttype) |false |string |Account type. `SPOT`|
|coin |false |string |Coin name |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|spot |Object| |
|> status |string |account status. `ACCOUNT_STATUS_NORMAL`: normal, `ACCOUNT_STATUS_UNSPECIFIED`: banned |
|> assets |array|Object |
|>> coin |string|Coin |
|>> frozen |string|Freeze amount |
|>> free |string|Free balance |
|>> withdraw |string|Amount in withdrawing |

---
