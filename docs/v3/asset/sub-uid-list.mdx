Query the sub UIDs under a main UID

:::caution
Can query by the master UID's api key **only**
:::

### HTTP Request
GET `/private/v1/asset/sub-member/member-ids`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|sub_user_id |array&lt;string&gt; |All sub UIDs under the main UID |

---

### Request Example

``` shell
GET /private/v1/asset/sub-member/member-ids HTTP/1.1
X-BAPI-API-KEY: 0b2XYJpi8bi9tZPhft
X-BAPI-SIGN: 520539314e551bb3b1cf6606b19e999906904fab9f766a0f709addadee558823
X-BAPI-TIMESTAMP: 1724600928448
X-BAPI-RECV-WINDOW: 1000000
Host: openapi-testnet.zoomex.com
```

### Response Example

```json
{
    "ret_code": 0,
    "ret_msg": "OK",
    "result": {
        "sub_user_id": [
            "103741450",
            "103747845"
        ]
    }
}
```
