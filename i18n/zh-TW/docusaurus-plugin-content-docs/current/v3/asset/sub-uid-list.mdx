查詢某個母帳戶的子帳號列表

:::caution
僅支持母帳號API key
:::

### HTTP 請求
GET `/private/v1/asset/sub-member/member-ids`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|sub_user_id |array&lt;string&gt; |所有子帳號 |
|transferable_sub_ids |array&lt;string&gt; |已經開啟了萬能劃轉的子帳號 |

---

### 請求示例

``` python
    def your_method(apiKey, secretKey):
        timestamp = int(time.time() * 10 ** 3)
        url = 'https://openapi-testnet.zoomex.com/private/v1/asset/sub-member/member-ids'
        method = "GET"
        params = {
            "api_key": apiKey,
            "timestamp": str(timestamp),
            "recv_window": "5000",
        }
        param_str = ''
        for key in sorted(params.keys()):
            v = params[key]
            if isinstance(params[key], bool):
                if params[key]:
                    v = 'true'
                else:
                    v = 'false'
            param_str += key + '=' + v + '&'
        param_str = param_str[:-1]
        hash = hmac.new(secretKey, param_str.encode("utf-8"), hashlib.sha256)
        signature = hash.hexdigest()
        sign_real = {
            "sign": signature
        }
        param_str = quote_plus(param_str, safe="=&")
        full_param_str = f"{param_str}&sign={sign_real['sign']}"
        urllib3.disable_warnings()
        s = requests.session()
        s.keep_alive = False
        response = requests.request(method, f"{url}?{full_param_str}", verify=False)

    def main():
        apiKey = "Your key"
        secret = "Your secret"
        your_method(apiKey, secret)


```
