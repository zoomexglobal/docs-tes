查詢某個母帳戶的子帳號列表

:::caution
僅支持母帳號API key
:::

### HTTP 請求
GET `/private/v1/asset/sub-member/member-ids`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|sub_user_id |array&lt;string&gt; |所有子帳號 |
|transferable_sub_ids |array&lt;string&gt; |已經開啟了萬能劃轉的子帳號 |

---

``` shell
GET /private/v1/asset/sub-member/member-ids?api_key=dVuOdOe0YxIzByQ8tV&sign=c8e474a329c4715bd7b1b80841fa49baf0ef2cbb8373fd731c5ed0181620400a&timestamp=1690947046394 HTTP/1.1
Host: openapi-testnet.zoomex.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1670988271299
X-BAPI-RECV-WINDOW: 50000

```
