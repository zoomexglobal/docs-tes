觸發子母劃轉動作

:::tip
請註意該接口入參請求數據中的transfer_id字段為UUID全局唯一。
該接口只能用於母子賬號之間的劃轉。
一個子賬號只會歸屬給一個母賬號。
:::

### HTTP 請求
POST `/private/v1/asset/sub-member/transfer`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|transfer_id |**true** |string |[UUID](https://www.uuidgenerator.net/dev-corner). 全局唯一 |
|coin |**true** |string |币种 |
|amount |**true** |string |划转金额 |
|sub_user_id |**true** |int32 |子账户uid |
|[type](../asset/enum#transfertype-type) |**true** |string |资金划转方向 |

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|transfer_id |string|划转请求的UUID |

---

### 請求示例

``` python
import time
import hmac
import hashlib
import uuid
import requests
import json

def create_transfer_request(apiKey, secret, body):
    url = 'https://openapi-testnet.zoomex.com/private/v1/asset/sub-member/transfer'
    timestamp = int(time.time() * 1000)
    recv_window = 1000000

    body['api_key'] = apiKey
    body['timestamp'] = timestamp
    body['recv_window'] = recv_window

    ordered_params = '&'.join([f"{key}={body[key]}" for key in sorted(body.keys())])

    signature = hmac.new(secret.encode('utf-8'), ordered_params.encode('utf-8'), hashlib.sha256).hexdigest()

    body['sign'] = signature

    headers = {
        'X-BAPI-API-KEY': apiKey,
        'X-BAPI-SIGN': signature,
        'X-BAPI-TIMESTAMP': str(timestamp),
        'X-BAPI-RECV-WINDOW': str(recv_window),
        'Content-Type': 'application/json'
    }

    response = requests.post(url, headers=headers, data=json.dumps(body))

    print("响应状态:", response.status_code)
    print("响应信息:", response.json())
    print("响应时间:", response.elapsed.total_seconds())

apiKey = 'your api key'
secret = 'your secret'
transferId = uuid.uuid4().hex
body = {
    "transfer_id":transferId,
    "coin":"USDT",
    "amount":"1",
    "sub_user_id":"162119745",
    "type":"OUT"
}

create_transfer_request(apiKey, secret, body)

```
