觸發子母劃轉動作

:::tip
請註意該接口入參請求數據中的transfer_id字段為UUID全局唯一。
該接口只能用於母子賬號之間的劃轉。
一個子賬號只會歸屬給一個母賬號。
:::

### HTTP 請求
POST `/private/v1/asset/sub-member/transfer`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|transfer_id |**true** |string |[UUID](https://www.uuidgenerator.net/dev-corner). 全局唯一 |
|coin |**true** |string |币种 |
|amount |**true** |string |划转金额 |
|sub_user_id |**true** |int32 |子账户uid |
|[type](../asset/enum#transfertype-type) |**true** |string |资金划转方向 |

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|transfer_id |string|划转请求的UUID |

---

### 請求示例

``` python
def your_method(apiKey, secretKey):
    timestamp = int(time.time() * 10 ** 3)
    url = 'https://openapi-testnet.zoomex.com/private/v1/asset/sub-member/transfer'
    method = "POST"
    params = {
        "api_key": apiKey,
        "timestamp": str(timestamp),
        "recv_window": "5000",
        "amount": "20",
        "sub_user_id": "1524613",
        "transfer_id": "d1fc12d2-c0c1-4954-8ca0-09d3343a246a",
        "type": "IN",
        "timestamp": "1690945245201",
        "coin": "USDT"
    }
    param_str = ''
    for key in sorted(params.keys()):
        v = params[key]
        if isinstance(params[key], bool):
            if params[key]:
                v = 'true'
            else:
                v = 'false'
        param_str += key + '=' + v + '&'
    param_str = param_str[:-1]
    hash = hmac.new(secretKey, param_str.encode("utf-8"), hashlib.sha256)
    signature = hash.hexdigest()
    sign_real = {
        "sign": signature
    }
    param_str = quote_plus(param_str, safe="=&")
    full_param_str = f"{param_str}&sign={sign_real['sign']}"
    urllib3.disable_warnings()
    s = requests.session()
    s.keep_alive = False
    response = requests.request(method, f"{url}?{full_param_str}", verify=False)

def main():
    apiKey = "Your key"
    secret = b"Your secret"
    your_method(apiKey, secret)

```
