獲取Zoomex服務器時間

### HTTP 請求
GET `/cloud/trade/v3/market/time`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| timeSecond | string | Zoomex服務器時間戳 (秒)  |
| timeNano | string | Zoomex 服務器時間戳 (微秒)  |

### 請求示例
```shell
GET /cloud/trade/v3/market/time HTTP/1.1
Host: openapi.zoomex.com
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "timeSecond": "1688639403",
        "timeNano": "1688639403423213947"
    },
    "retExtInfo": {},
    "time": 1688639403423
}
```
