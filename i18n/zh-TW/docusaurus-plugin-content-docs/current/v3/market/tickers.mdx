可獲取到快照的最新市場價格，最佳買賣價格，以及過去時間內的交易量等.
> **覆蓋範圍:  USDT永續 / 反向合約**

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP請求
GET `/cloud/trade/v3/market/tickers`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |產品類型. `linear`,`inverse` |
|[symbol](../enum#symbol) |false |string |合約名稱 |

### 請求示例

```shell
GET /cloud/trade/v3/market/tickers?category=inverse&symbol=BTCUSD HTTP/1.1
Host: openapi-testnet.zoomex.com
```
