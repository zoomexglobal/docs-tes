### HTTP 請求

GET `/contract/v3/private/copytrading/position/close`

### 請求參數

| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[symbol](../enum#symbol) |**true** |string |合約類型 |
|[positionIdx](../enum#positionidx) |**true** |string |Position idx, 用於在不同倉位模式下標識倉位：0-單向持倉; 1-雙向持倉Buy; 2-雙向持倉Sell|

### 返回參數

無

### 請求示例

``` shell
curl --location --request POST 'https://openapi-testnet.zoomex.com/cloud/contract/v3/private/copytrading/position/close' \
--header 'Content-Type: application/json' \
--data-raw '{"symbol":"BTCUSDT","positionIdx":2,"api_key":"XXXXX","timestamp":1689850686341,"recv_window":5000,"sign":"XXXXX"}'
```
