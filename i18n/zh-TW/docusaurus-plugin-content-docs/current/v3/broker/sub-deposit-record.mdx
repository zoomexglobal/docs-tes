> API頻率: 300次/分

### HTTP 請求
GET `/private/v1/broker/asset/query-member-deposit-record`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|memberId |**true** |string |直客UID|
|coin |false |string |幣種 |
|startTime |false |integer |開始時間戳 (秒) |
|endTime |false |integer |結束時間戳 (秒) |
|limit |false |integer |每頁數量限制. [`1`, `50`]. 默認: `20`|
|cursor |false |string |游標，用於翻頁|

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|rows |array|Object |
|> coin |string |幣種 |
|> chain |string|鏈名 |
|> amount |string|充值金額 |
|> tx_id |string|交易Id. 充值失敗或取消充值時為空 |
|> status |integer|充值狀態:0-充值中 1-充值成功 2-充值失败 |
|> to_address |string|充值的目標地址 |
|> tag |string|充值目標地址的tag |
|> success_at |string|最後更新時間 |
|> confirmations |string|确认区块的数量 |
|> tx_index |string|交易序列号 |
|> block_hash |string|鏈上的哈希數 |
|cursor |string |游標，用於翻頁|

### 請求示例

```python
       def your_method(apiKey, secretKey):
           timestamp = int(time.time() * 10 ** 3)
           url = 'https://openapi-testnet.zoomex.com/private/v1/broker/asset/query-member-deposit-record'
           method = "GET"
           params = {
               "api_key": apiKey,
               "timestamp": str(timestamp),
               "recv_window": "5000",
               "memberId": 101479471,
           }
           param_str = ''
           for key in sorted(params.keys()):
               v = params[key]
               if isinstance(params[key], bool):
                   if params[key]:
                       v = 'true'
                   else:
                       v = 'false'
               param_str += key + '=' + v + '&'
           param_str = param_str[:-1]
           hash = hmac.new(secretKey, param_str.encode("utf-8"), hashlib.sha256)
           signature = hash.hexdigest()
           sign_real = {
               "sign": signature
           }
           param_str = quote_plus(param_str, safe="=&")
           full_param_str = f"{param_str}&sign={sign_real['sign']}"
           urllib3.disable_warnings()
           s = requests.session()
           s.keep_alive = False
           response = requests.request(method, f"{url}?{full_param_str}", verify=False)

       def main():
           apiKey = "Your key"
           secret = b"Your secret"
           your_method(apiKey, secret)

```
