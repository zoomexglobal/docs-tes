> API頻率: 300次/分

### HTTP 請求
GET `/private/v1/broker/member-info`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|uid |**true** |string |直客UID|

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|is_valid |integer| 是否直客，0：否；1：是|
|signup_time |string| 成為直客時間|

### 請求示例

```python
       def your_method(apiKey, secretKey):
           timestamp = int(time.time() * 10 ** 3)
           url = 'https://openapi-testnet.zoomex.com/private/v1/broker/member-info'
           method = "GET"
           params = {
               "api_key": apiKey,
               "timestamp": str(timestamp),
               "recv_window": "5000",
               "uid": 101479471,
           }
           param_str = ''
           for key in sorted(params.keys()):
               v = params[key]
               if isinstance(params[key], bool):
                   if params[key]:
                       v = 'true'
                   else:
                       v = 'false'
               param_str += key + '=' + v + '&'
           param_str = param_str[:-1]
           hash = hmac.new(secretKey, param_str.encode("utf-8"), hashlib.sha256)
           signature = hash.hexdigest()
           sign_real = {
               "sign": signature
           }
           param_str = quote_plus(param_str, safe="=&")
           full_param_str = f"{param_str}&sign={sign_real['sign']}"
           urllib3.disable_warnings()
           s = requests.session()
           s.keep_alive = False
           response = requests.request(method, f"{url}?{full_param_str}", verify=False)

       def main():
           apiKey = "Your key"
           secret = "Your secret"
           your_method(apiKey, secret)

```
